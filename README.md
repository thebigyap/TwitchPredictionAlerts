# Twitch Prediction Alerts (Microsoft Edge Extension)

A microsoft/chrome extension that plays a sound when a twitch streamer starts a prediction. This extension runs on `twitch.tv`, detects likely "Prediction started" UI events, and alerts the user of them via a sound que. As a bonus, auto-claims the green Channel Points bonus chest.

## Load in Edge

OPTION 1. Edge/Chrome Extension:
1. Go to the edge extension page (URL TBD)
2. Click Add/Install

OPTION 2. Manually installing:
1. Download repository
2. Extract zip
3. Open `edge://extensions`
4. Turn on **Developer mode**
5. Click **Load unpacked**
6. Select the extracted folder
